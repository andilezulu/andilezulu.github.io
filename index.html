for building beautiful, natively compiled, multi-platform applications from a single codebase.”

“Flutter transforms the app development process. Build, test, and deploy beautiful
mobile (iOS and Android), web, desktop, and embedded apps from a single codebase.”

And WTF is Kotlin then?

Kotlin is a modern but already mature programming language aimed to make developers happier.
It’s concise, safe, interoperable with Java and other languages, and provides
many ways to reuse code between multiple platforms for productive programming.

Kotlin is a great fit for developing server-side applications. It allows you to write concise
and expressive code while maintaining full compatibility with existing Java-based technology stacks, all with a smooth learning curve - JetBrains
